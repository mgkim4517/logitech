## Publishing Guide
```
1. 통일된 식별자명을 위해 케밥 표기법 사용
ex) logitech-container

2. 주석
/***** header, main, footer *****/, <!-- ***** header, main, footer ***** -->
/*** section ***/, <!-- *** section *** -->
/* 그 외 */, <!-- * 그 외 * -->

3. class명 표기법
article > container > wrapper > content > box
```
